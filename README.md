## Instalar dependencias

```bash
$ npm install --force
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Levantar la base de datos

Utilizamos docker compose `docker compose up -d`

## Stack usado
* MongoDB
* Nest
